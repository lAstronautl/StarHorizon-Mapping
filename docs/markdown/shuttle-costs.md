# Цена шаттлов

Есть формула с помощью которой выставляется цена для шаттлов. Бывают моменты в которых может быть изменена цена не по формуле ради баланса шаттла. Но в основном она используется для каждой оценки шаттлов и не меняется.

### Пример из стандартного прототипа шаттла 
```yml
price: 16000 # 14400, наценка верфи 10% = 15840
```

### Подсчёт цены

Берём стандартную цену в 14400 далее умножаем на 1.10 (“10” это процент наценки иза тира).

Выходит цена после наценки: 15840

И просто округляем до красивой цены: 16000

Вот так и происходит магия подсчета цены!
