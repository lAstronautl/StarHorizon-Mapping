# Приёмная верфь

### Это дискорд [канал](https://discord.com/channels/1348258341658562560/1398255094411165696) который позволяет обычным игрокам выкладывать свои шаттлы которые могут быть добавлены на основной сервер после проверки

## Форма публикации шаттлов
### Имя шаттла (заголовок)
```
Описание:
id шаттла:
Цена:
Корпорация которая сделала шаттл: (по умолчанию NRZN)
Размер:
Доступен в:
Тип:
Генераторы:
Дополнительные слова (если хотите)
Скриншоты или рендер
```
## Пример формы
### Barge
```
Описание: Средний карго шаттл, переоборудованное в утилизаторский бар.
id шаттла: Barge (это НЕ название шаттла а его id который вы даёте в BecomesStation.)
Цена: 56000 # После оценки 42901, наценка ~30%
Корпорация которая сделала шаттл: NRZN
Размер: Средний
Доступен в: верфи фронтира
Тип: Карго, Утилизаторский
Генераторы: Солнечные панели, плазменный генератор
Удобный карго шаттл для перевозок и так же имеет хороший бар
```
## Форма под публикацию аванпостов
### Имя аванпоста (заголовок)
```
Описание:
Минимальная дистанция спавна:
Максимальная дистанция спавна:
Должности:
Транзитный шаттл:
Скриншоты и карта аванпоста
```
## Пример формы
### Courthouse
```
Описание: Суд для вынесения наказаний игрокам
Минимальная дистанция спавна: 1150
Максимальная дистанция спавна: 2050
Должности: Судья 1, Офицер ДФСБ 3
Транзитный шаттл: Есть
```
## Форма публикации ремапов
### Имя шаттла или аванпоста который ремапнут

```
Изменения:
Скриншоты изменений
```

---
### Также есть несколько примечаний

1. Заголовок должен содержать имя шаттла или аванпоста.  
2. Прототип шаттла/аванпоста не обязателен. Но тогда заполнение всех пунктов обязательно.  
3. Создатели шаттлов/аванпостов не считая фракций могут получить от 100к до 500к игровой валюты за одобрение их шаттла на сервер.  
4. Вы можете использовать этот канал для поиска людей, которые помогут с маппингом шаттла, или как девблог по разработке шаттла.  
5. Старайтесь заливать финальный вариант шаттла, изменения во время тестирования нежелательны. Если не готовы то ставьте тег “Проектируется”.
